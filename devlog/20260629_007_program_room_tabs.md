## 한 일

### 프로그램 페이지 탭 위젯
- 같은 시간대에 병행되는 룸(세션)을 세로 나열 대신 **탭으로 전환**.
- 일자 선택 아래 가로 스크롤 탭바(Plenary + 룸 축약명: `Int'l I/II/III`, `773/775/776`).
- 룸 패널은 한 번에 하나만 표시(`.panel.on`). 룸 헤더에 층(예: `· 7F`) 병기.
- 마지막으로 본 룸 이름을 `sessionStorage`에 저장 → 일자 변경 시 동일 룸 유지(없으면 첫 탭).
- **서버에서 기본 활성 탭/패널 지정**(plenary 있으면 plenary, 없으면 첫 룸) → JS 실패해도 첫 패널 노출.
- 뷰에 `short_room()` 추가(룸 축약 라벨).
