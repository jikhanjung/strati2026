# 024. Program: 플로팅 날짜/룸 + 현재시각 줄에 안내 + 날짜칩 축소

- 날짜: 2026-06-30
- 상태: 완료(테스트 서버 0.1.17 반영 예정).

## 한 일 (`program.html`, `style.css`)

### 1) 스크롤 시 날짜/룸 플로팅
- 날짜 버튼(daybar)·룸 탭(roomtabs)이 위로 스크롤돼 안 보이면, 앱바 바로 아래에
  **현재 날짜 + 룸**을 `#ctxbar`(네이비 바)로 플로팅.
- 판정: `roomtabs.getBoundingClientRect().bottom <= appbar.bottom` 이면 노출.
  스크롤·리사이즈·탭전환 시 갱신. 룸 이름은 활성 탭 텍스트에서 가져옴.

### 2) "진행 중 발표 없음" 메시지를 현재 시각 줄로
- 기존: 상단 배너로 표시. 변경: **현재 시각 빨간 줄**에 라벨로 표시.
  - 줄 위치가 곧 메시지 위치 → 아침(발표 전)=첫 발표 위, 저녁(종료 후)=마지막 발표 아래.
- 상단 배너(`#nowbanner`)는 **오늘이 학회일이 아닐 때**("No talks are scheduled today.")만.

### 3) 날짜 칩 높이 축소
- `.daychip`을 2줄(column)에서 **1줄(가로)** 로: "Mon Jun 29"가 한 줄. 패딩·폰트도 소폭 축소.

## 검증

- 렌더 JS `node --check` 통과, `#ctxbar`·`ctxday`("Tue Jun 30") 렌더 확인.

## 메모

- ctxbar z-index 19(앱바 20 아래), top=앱바 bottom으로 겹치지 않게 배치.
- 다른 날짜(비-오늘)를 보는 중엔 줄/메시지 없음(브라우징으로 간주).
