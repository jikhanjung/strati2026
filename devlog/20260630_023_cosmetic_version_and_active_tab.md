# 023. Cosmetic: 버전 위치 + 활성 탭 표시

- 날짜: 2026-06-30
- 상태: 완료(테스트 서버 0.1.16 반영 예정).

## 한 일

### 버전 번호 위치 (`base.html`)
- 헤더 우측 별도 칩이던 `vX.Y.Z`를 **"Unofficial Bookmarks" 바로 뒤**(brand-sub 안)로 이동.

### 활성 탭 강조 (`style.css`)
- 하단 탭바에서 현재 페이지가 잘 안 보였음(색만 회색→네이비).
- `.tab.on`에 **상단 액센트 바(3px, 네이비) + 옅은 배경 틴트** 추가로 명확히.
- 활성 판정 로직(`{% if u == 'program' %}…`)은 기존 그대로(Sessions는 session_detail 포함).

## 검증

- 헤더에 `v0.1.15`가 brand-sub 내부 렌더 확인.
- 각 페이지에서 해당 탭에 `class="tab on"` 출력 확인.
