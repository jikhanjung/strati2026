# 020. My Plan: 현재 시각 줄/버튼 + 겹치는 발표 컬럼 표시

- 날짜: 2026-06-30
- 상태: 완료(테스트 서버 0.1.14 반영 예정).

## 한 일 (`timetable.html`, `style.css`)

### 현재 시각 줄 + 점프 버튼
- 프로그램 페이지와 동일한 빨간 줄 + `● Now` 플로팅 버튼을 My Plan에도.
- 줄은 **오늘 날짜 섹션**(섹션에 `data-date` 부여)에만 표시.
- 위치 계산 `nowY`는 **getBoundingClientRect 기반**으로 작성 — 아래의 그리드
  절대배치(컬럼)에서도 카드 실측 위치로 정확히 보간됨. 진행 중 카드 안 시간비례,
  빈틈은 직전/다음 카드 경계 보간.
- 첫 로드 1회 자동 스크롤(firstPlace), 1분 갱신, 스크롤 시 버튼 노출.

### 시간 겹치는 북마크 → 비례 컬럼 그리드
- 같은 날 발표를 **겹침 클러스터**로 묶음(running max-end). 단독 발표는 기존처럼 전체폭 카드.
- 클러스터는 **비례 타임그리드**: 박스 높이=발표 길이, 세로 위치=시작 시각,
  컬럼=동시 발표(greedy 배정, 겹치지 않으면 컬럼 재사용).
- 2개 이상 동시면 각 카드 폭을 **86%** 로 줄여 옆 카드가 살짝 보이게 + **좌우 스크롤**
  (`.cluster{display:flex;overflow-x:auto}`, `.gcol{flex:0 0 86%}`). 4개 이상도 같은 방식으로 스크롤.
- 박스 최소 높이 50px, 분당 3.4px. 길이가 짧고 제목이 길면 잘릴 수 있음(overflow hidden) — ① 방식 선택에 따른 트레이드오프.
