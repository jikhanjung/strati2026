# 016. 현재 시각 줄 첫 로드 위치 버그 수정 + 안내 문구 영어화

- 날짜: 2026-06-30
- 상태: 완료(테스트 서버 0.1.10 반영, git/이미지 푸시).

## 배경

014의 현재 시각 빨간 줄이 **페이지 첫 로드 시에만** 첫 발표 위가 아니라
첫·둘째 발표 사이로 내려가는 현상 보고. 또한 015의 안내 배너/버튼 문구를 영어로.

## 원인

`placeNow()`에서 `nowY()`가 카드의 `offsetTop`을 읽는 시점에 활성 패널이 아직
`position: relative`가 아니었음(설정이 측정 *이후* 줄을 붙이기 직전에 있었음).
→ 카드의 offsetParent가 패널이 아닌 바깥 상위 요소가 되어 daybar·룸탭 높이만큼
큰 offsetTop이 측정됨. 그 값을 패널 내부 좌표로 사용해 줄을 붙이니 아래로 밀림.
탭 전환·리사이즈 때는 직전 호출에서 이미 relative가 걸려 정상화 → "첫 로드에서만" 발현.

## 한 일 (`program.html`)

- `panel.style.position = 'relative'`를 `nowY()` 측정 **전**으로 이동
  (첫 렌더부터 패널이 offsetParent가 되도록).
- 안내 문구 영어화:
  - "오늘은 예정된 학회 발표가 없습니다." → "No talks are scheduled today."
  - "지금은 진행 중인 발표가 없습니다." → "No talk is in session right now."
  - 점프 버튼 `● 현재 시각` → `● Now`.

## 검증

- 렌더 JS `node --check` 통과, 영어 문구/버튼 출력 확인.
- 테스트 서버 v0.1.10 응답 확인.

## 메모

- CSS로 `.panel { position: relative }`를 거는 대안도 있었으나, 활성 패널에만
  JS로 거는 기존 방식을 유지(다른 패널 스택에 영향 없음).
